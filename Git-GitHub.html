Git

  • Git is the most commonly used version control system
  • Git tracks changes you maketo files so you have a record of what has been done
  • Git lets you revert back to specific versions of files if needed
  • Git helps to make collaboration efforts easier by allowing files to be merged into one source

GitHub

  • GitHub is a Git repository hosting service, with many of its own features
  • GitHub is a Web-based graphical interface
  • Github is a cloud based service, it allows you to store files in a safe place off of your local drive
